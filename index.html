웹 퍼블리셔 박승주입니다.:)

1년 8개월차 웹 퍼블리셔입니다.

효율적인 UI/UX가 무어일지 계속 고민하고, 웹 접근성을 고려한 마크업을 하기 위해 노력합니다. 모든 사람이 사옹하기에 편리하고, 즐거운 사이트를 만드는 것이 제가 하는 일이라고 생각합니다.

더 나아가 CSS와 JS를 활용한 동적모션과 서버 통신 후 데이터를 가져와 화면에 나타낼 떄 원하던 화면을 구현해 낼 때 성취감을 얻습니다.

사용자의 편의성이 높은 사이트를 만들어 보람을 얻고, 더 나은 코드를 작성하기 위해 끊임없이 공부하며 발전하고, 많은 시행착오를 겪고 결국엔 이뤄내며 얻는 기쁨과 성취감이 제가 개발을 좋아하는 이유입니다.
